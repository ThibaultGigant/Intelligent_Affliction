override int Nourriture.consome ( bool  flag)

Consommation journalière de nourriture par la population.

Paramètres
fragIndique si l'on veut une consomation effective, ou seulement une estimation de la consommation
Renvoie
Si flag est vrai, retourne le nombre d'insatisfaits, sinon, retourne le montant à consommer

Chaque personne mange environ une "ressource" [taille de la population] * x où x est choisi aléatoirement dans [0.95,1.05]
